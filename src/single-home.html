Наружные стены первого этажа выполняются из материала, выбранного заказчиком для строительства (цельный брус, клееный брус, оцилиндрованное бревно).

Внутренние перегородки первого этажа - каркасные, обшиваются вагонкой класса "В" с двух сторон.

Каркасные перегородки устанавливаются с осадочным зазором для возможности усадки стен дома.

Утепление внутренних перегородок не производится.

Каркас наружных стен выполняется из обрезной доски 50х150 (толщина) мм.

Каркас внутренних перегородок выполняется из обрезной доски 50х100 (толщина) мм.

Наружные стены и внутренние перегородки обшиваются вагонкой класса "В" с двух сторон.
